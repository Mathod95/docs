## Bucket

=== "Bucket with User Defined Values"

    ```yaml linenums="1" title="Bucket with User Defined Values"
    apiVersion: s3.aws.m.upbound.io/v1beta1
    kind: Bucket
    metadata:
      namespace: NAMESPACE
      name: APPNAME-SERVICE-ENVIRONMENT-bucket
    spec:
      forProvider:
        region: REGION
        tags:
          company: COMPANY
          appName: APPNAME
          service: SERVICE
          environment: ENVIRONMENT
          team: TEAM
          kind: bucket
          region: REGION
          account: ACCOUNT_ID
          createdBy: mathod
          createdAt: 2026-06-01
          provisionedBy: crossplane
      providerConfigRef:
        name: default
        kind: ProviderConfig
    ```

## BucketServerSideEncryptionConfiguration

```yaml linenums="1"
apiVersion: s3.aws.m.upbound.io/v1beta1
kind: BucketServerSideEncryptionConfiguration
metadata:
  namespace: dev-environment
  name: my-secure-bucket-encryption
spec:
  forProvider:
    bucket: my-secure-bucket
    rules:
      - applyServerSideEncryptionByDefault:
          sseAlgorithm: AES256
    tags:
      kind: BucketServerSideEncryptionConfiguration
  providerConfigRef:
    name: aws-provider
    kind: ProviderConfig
```
